# 0x0D. C - Preprocessor

The aim of this project is to learn about the C program Compilation Process, object-like macros, macro arguments and include guard in **C language**.

## Technologies
* Scripts written in Bash 5.0.17(1)
* Tested on Ubuntu 20.04 LTS
* Vi/Vim/Emacs Editors

## Files
Scripts written in C:

| Filename | Description |
| -------- | ----------- |
| `0-object_like_macro.h` | Header file that defines a macro named `SIZE` |
| `1-pi.h` | Header file that defines a macro named `PI` |
| `2-main.c` | Prints the name of the file it was compiled from |
| `3-function_like_macro.h` | Function-like macro `ABS(x)` that computes the absolute value of a number `x` |
| `4-sum.h` | Function-like macro `SUM(x,y)` that computes the sum of the numbers `x` and `y` |
| `101-preprocessor_abuse.c` | Prints `Hello` |
